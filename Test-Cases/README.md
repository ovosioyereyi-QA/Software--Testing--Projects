# Test Cases

This folder contains sample manual test cases created to demonstrate my ability to design, organize, and execute software tests.

The test cases cover common web application features such as:

- User Login
- User Registration
- Search Functionality
- Add to Cart
- Checkout
- Password Reset

Each test case includes:

- Test Case ID
- Test Scenario
- Preconditions
- Test Steps
- Test Data (where applicable)
- Expected Result
- Actual Result (where applicable)
- Status (Pass/Fail)
